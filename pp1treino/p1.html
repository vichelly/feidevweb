Introdução

Com o HTML você descreve o conteúdo da página, enquanto o CSS aplica o estilo descrevendo como o conteúdo deverá aparecer. Fontes, cores, imagens de fundo, espaçamento entre linhas, layout de página e assim por diante são todos controlados com CSS. Você pode até adicionar efeitos especiais e animação básica à sua página (Fonte Usada: SansSerif, tamanho 20pt)

Juntamente com HTML e CSS, o JavaScript é uma da três principais tecnologias da World Wide Web, Javascript permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos da web (Fonte Usada: SansSerif, tamanho 20pt)
